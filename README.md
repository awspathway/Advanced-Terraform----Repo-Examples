# Advanced Terraform Learning Path

A structured learning repository for mastering Infrastructure as Code with Terraform and AWS.

## Repository Structure

```
aws pathway-sample/
├── Advanced Terraform/
│   └── Terraform-Basic/
│       ├── IAC/
│       │   ├── assets/
│       │   └── intro.mdx          # Infrastructure as Code fundamentals
│       └── Terraform-Fundamentals/
│           ├── assets/
│           └── intro.mdx          # Terraform core concepts
└── README.md
```

## Learning Topics

### Infrastructure as Code (IaC)
- Core IaC concepts and benefits
- Popular tools comparison
- Best practices and workflows
- Interactive quiz and flashcards

### Terraform Fundamentals
- Terraform architecture and components
- HCL configuration language
- Basic workflow and commands
- Hands-on exercises

## Getting Started

1. Start with **IAC/intro.mdx** for Infrastructure as Code fundamentals
2. Progress to **Terraform-Fundamentals/intro.mdx** for Terraform specifics
3. Complete quizzes and review flashcards for reinforcement

## Features

- Interactive quizzes for knowledge validation
- Flashcard summaries for quick revision
- Structured learning path
- Practical examples and exercises
