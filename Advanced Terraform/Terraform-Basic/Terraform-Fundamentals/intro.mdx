# Terraform Fundamentals

Terraform is an open-source Infrastructure as Code tool that allows you to define and provision infrastructure using a declarative configuration language.

## Core Concepts

- **Providers**: Plugins that interact with APIs of cloud platforms
- **Resources**: Infrastructure components like EC2 instances, VPCs
- **State**: Terraform's record of your infrastructure
- **Configuration**: HCL files defining your infrastructure

## Basic Workflow

1. **Write** - Define infrastructure in .tf files
2. **Plan** - Preview changes with `terraform plan`
3. **Apply** - Create infrastructure with `terraform apply`
4. **Destroy** - Remove infrastructure with `terraform destroy`
